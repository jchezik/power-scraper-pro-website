Frequently asked questions.

Everything you need to know about Power Scraper Pro.

Power Scraper Pro retrieves metadata from TMDB (The Movie Database) for movies, TVDB for TV shows, and Fanart.TV for high-resolution artwork. All three services are industry standards used by professional media centers worldwide. Your TMDB API key is required for movie metadata.

Yes. Power Scraper Pro generates NFO files and organizes media in the exact folder structure that Kodi expects. After processing with Power Scraper Pro, simply point Kodi at your media folders and it will read all metadata locally without needing to scrape online sources again.

Failed items are clearly marked in the interface. You can manually search TMDB or TVDB directly from the app, select the correct match, and retry scraping with one click. The dashboard also shows TMDB suggestions for items that might have close matches.

Power Scraper Pro renames files and organizes them into proper folder structures. The original video content is never modified, only renamed and moved. You can configure the output directory and naming format in settings. A preview is always shown before any changes are made.

Power Scraper Pro is built with Swift 6.2 using strict concurrency and SwiftUI. It leverages Metal compute shaders for GPU-accelerated thumbnail generation optimized specifically for M-series chips. These modern technologies deliver the best performance, reliability, and security.

Power Scraper Pro will be a one-time purchase with no subscription required. Early access pricing will be available for beta testers who sign up. Future major versions may be separate purchases, but all minor updates and bug fixes are included for life.

Yes. Configure any folder as a Watch Folder and Power Scraper Pro will automatically detect new media files, scan them, fetch metadata, and organize them into your library. This works for both movies and TV shows with real-time file system monitoring.

Session state is continuously saved to disk. If the app quits, crashes, or you need to stop, your progress is preserved. When you relaunch, Power Scraper Pro offers to resume exactly where you left off. No duplicate downloads or wasted API calls.
